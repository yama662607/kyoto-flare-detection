# Outputs and Debug Artifacts

This repository generates plot PDFs and debug CSV files when running notebooks.
To keep the repository clean, generated artifacts are stored under `outputs/` and are ignored by git.

## Output Directories

- `outputs/figures/`
  - Plot PDFs generated by notebooks (e.g., `flare_cumenergy_*.pdf`).
- `outputs/debug/`
  - Per-run debug CSV snapshots used for cross-environment comparison.
  - Each run writes to a timestamped subfolder: `outputs/debug/YYYYMMDD_HHMMSS/`.

## Debug CSV Formats

- `freq_*.csv`: `x_starspot_scaled,y_freq_per_day,yerr_per_day`
- `freq_fit_*.csv`: `x_fit,y_fit`
- `totalene_*.csv`: `x_starspot_scaled,y_total_energy_scaled`
- `totalene_fit_*.csv`: `x_fit,y_fit`
- `maxene_*.csv`: `x_starspot_scaled,y_max_energy_scaled`
- `maxene_pearson_*.csv`: `x_starspot_scaled,y_max_energy_scaled`
- `cumenergy_<star>_<sector>.csv`: `energy_erg,rate_per_day`

## Comparing Debug Outputs

Use the helper script to compare two debug folders:

```bash
python tools/compare_debug_outputs.py outputs/debug/20260206_120000 outputs/debug/20260206_121500
```

You can adjust tolerances with `--rtol` and `--atol` if needed.
